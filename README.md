# Creative-Project
Mouse Drawing App for Narayanan's Lab at the University of Iowa Neurology Department.

# Dependences
### Python packages
- Kivy
- numpy
- RPi.GPIO
### Arduino
- fed3 - [Install Instructions](https://github.com/KravitzLabDevices/FED3_library/wiki#installation-instructions)
